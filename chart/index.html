Plot Guide

Bar plot

  • Init a chart use Echart class
chart = Echart('Title', 'subtitle')
  • Use Bar class to init a series of bar plot
chart.use(Bar('Country', [2, 3, 4, 5]))
  • Use Axis to set the axis
chart.use(Axis('category', 'bottom', data=['Nov', 'Dec', 'Jan', 'Feb']))
  • Use plot method to view your bar plot, result is as same as above.

Scatter

  • class Scatter is used for ploting a scatter
# generate a random point
points = [[random.randint(0, 100) / 10.0, random.randint(0, 100) / 10.0] for _ in range(200)]
# set the random point as source data of scatter
chart.use(Scatter(name="points", data=points))
