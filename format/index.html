Format Discuss

Overview

We are trying to support main-stream pathway format: KGML, BioPAX, SBGN-PD and GMPL.

Choice

  • For the visualization purpose, we prefer more on the format which contains layout info. So the SBGN format is the first choice.
  • KGML and GPML are from KEGG and WikiPathway, both are human made and well desgined data. so we DO NOT convert it to BioPAX or SBGN intend to keep is original looking.
  • BioPAX, its visualization require additional layout procedure, so, if not necessary, we will use other format replace it.

Reactome
